## 🎯 Project Overview
This repository contains the source code for a static portfolio website hosted on GitHub Pages. It highlights my academic background, technical proficiencies, and practical experience in Software Quality Assurance (SQA).

## 🏗️ Content Structure
The website is organized into four core sections to provide a comprehensive view of my profile:
1. **About Me (Giới thiệu):** A brief introduction to my career goals and transition from Manual to Automation Testing.
2. **Projects (Dự án):** Detailed showcases of my recent work, including an E2E Automation Testing framework with Cypress & GitHub Actions.
3. **Certificates (Chứng chỉ):** Professional certifications obtained from platforms like Coursera and Mendix Academy.
4. **Contact (Liên hệ):** Professional links to reach out via Email, LinkedIn, and GitHub.
